# 🔄 LIMPAR CACHE DO NAVEGADOR

O deploy foi atualizado, mas seu navegador pode estar usando cache antigo. **Siga estes passos:**

---

## 🔧 **MÉTODO 1: Limpar Cache (Windows)**

### Chrome / Edge / Firefox
1. Pressione: **Ctrl + Shift + Delete**
2. Marque: "Todos os tempos"
3. Marque: "Imagens em cache"
4. Marque: "Cookies e outros dados do site"
5. Clique: **LIMPAR DADOS**
6. **Recarregue a página:** F5

### Safari (Mac)
1. Menu: **Desenvolvimento** → **Vazio o Cache**

---

- **Windows:** Ctrl + Shift + N
